# UI/UX — To-Do List 1.0

## Paleta de Cores

* **Primária (#F79E46):** Botões principais
* **Secundária (#240115):** Seções principais
* **Terciária (#2471b4):** Outros botões
* **Fundo (#FFBFA0)**
* **Neutralizador (#CEC3C1)**
* **Texto (#FFF)**

---

## Mobile First

* Layout centralizado
* Header com título Tarefas.
* Campo de adicionar tarefa logo abaixo do header.
* Filtros abaixo do campo de adição.
* Estastísticas com número de tarefas pendentes e feitas.
* Lista única de tarefas (pendentes e concluídas) controlada por estado.

---

## Desktop

* Mesma estrutura do mobile.
* Header permanece com título e navegação visível.
* Cards com largura limitada para evitar telas muito largas.

---

## Comportamentos

* Hover e foco usam a cor variada dark.
* Ordem padrão: pendentes primeiro, concluídas depois, via ordenação no JS.
